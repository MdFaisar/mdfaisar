## 📂 Featured Projects

### 🔐 1. PixelMind — Image-Based Text Encryption System
> `Jan 2025 – May 2025` &nbsp;|&nbsp; **[View on GitHub →](https://github.com/MdFaisar)**

**Tech Stack:** Python · Flask · MongoDB · JavaScript · HTML/CSS

A novel approach to secure file sharing — PixelMind converts plain text files (`.txt`, `.md`, `.py`, `.c`, etc.) into pixelated PDF images using a custom encryption algorithm, making the data visually unreadable and safely shareable.

- 🔒 Custom encryption algorithm that transforms text into pixel-art PDFs
- 📄 Supports multiple text file formats for flexible use
- 🤖 Integrated AI chatbot for intuitive user navigation
- 📥 Download or share encrypted PDFs directly from the platform

> `Mar 2025 – Apr 2025` &nbsp;|&nbsp; **[View on GitHub →](https://github.com/MdFaisar)**

**Tech Stack:** Python · Flask · Firebase · JavaScript · HTML/CSS

An AI-driven career guidance platform that recommends personalized career paths based on a user's interests, education background, and work experience — all gathered through an engaging, gamified onboarding experience.

- 🎮 Gamified data collection for a delightful user experience
